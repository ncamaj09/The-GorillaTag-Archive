Information

This is an archive of old versions of Gorilla Tag for the Oculus. The download files are APK files, which are Android Packages similar to EXE files on Windows. These APK files are not viruses and are 100% real. This project was created to allow users to experience older versions of Gorilla Tag instead of only playing the latest versions.
